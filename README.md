# Lista de Pendientes 2026

Aplicación web desarrollada con **Django** para gestionar tareas pendientes, con autenticación de usuarios y diseño moderno.

---

## Funcionalidades

- Registro y login de usuarios
- Crear, editar, eliminar y completar tareas
- Cada usuario gestiona sus propias tareas
- Diseño moderno y responsivo con CSS
- Protección de rutas con autenticación
